Introdução

O Rocket.chat é a solução para melhorar a produtividade da Prefeitura Municipal de Santarém, garantindo uma comunicação ágil, segura e em tempo real. A plataforma é muito versátil com recursos como compartilhamento de arquivos e chamadas de videoconferência. Além de otimizar a comunicação interna, é possível melhorar a comunicação com a população, através de um Livechat e a possibilidade de criação de um canal de comunicação. Em tempos de Pandemia e a necessidade de se evitar aglomeração, esta plataforma torna-se necessária.

Ambiente do Usuário

Ativar/Desativar Livechat

Este botão serve para Ativar ou Desativar a visibilidade do usuário para o atendimento pelo Livechat. Estando habilitado, o usuário fica disponível na fila para atender os usuários pelo Livechat. Estando desabilitado, o usuário fica indisponível para atender os usuários pelo Livechat.

Informações do Visitante

Este menu é utilizado para visualizar as informações do visitante, quando há um cadastro feito anteriormente no sistema, mostrado na próxima seção, assim como o botão editar, fechar e encaminhar.

Editar/Cadastrar novo visitante

Este menu tem a finalidade de cadastrar o contribuinte que procura atendimento. Para isso, você precisa clicar em "Informações do Visitante", e em seguida no botão "Editar".

Imagem da janela de bate papo inteira, com os itens circulados


Estão disponíveis para cadastro:

  • Nome: Você pode pedir para o usuário informar o nome, e pode cadastrar neste campo.
  • E-mail: Você pode pedir para o usuário informar o e-mail, para posterior contato.
  • Phone: Vocẽ pode pedir para o usuário informar o telefone, para posterior contato.
  • Tópico: Você pode cadastrar sobre qual foi o assunto do atendimento ao contribuinte.
  • Tags: Vocẽ pode colocar uma etiqueta, para identificar o assunto do atendimento. Ex: IPTU, ITBI, CND.

    Encaminhar usuário (IMPORTANTE - somente com a conversa ativa)

    Este menu tem como objetivo encaminhar o contribuinte a um setor ou outro atendente específico correto.

    Ex: O contribuinte está em contato com um atendente da Divisão de Tributos, e necessita ser encaminhado para a Divisão de Cadastro Imobiliário.

    Para isto, deve-se clicar em "Informações do Usuário", em seguida clicar em "Encaminhar". Em seguida, deve-se escolher Encaminhar ao departamento, clicando em "Selecione um Departamento" e escolher para qual será encaminhado, ou Encaminhar ao usuário, clicando em "Selecione um usuário" e digitando o nome do usuário que efetuará o atendimento.

    Imagem da janela de bate papo inteira, com os itens ENCAMINHAR circulados

    Imagem de uma janela de bate papo em curso, de um encaminhamento realizado.


    Encerrar atendimento

    Este menu é utilizado para encerrar/finalizar o atendimento ao contribuinte, para que a sessão seja fechada. Este menu só está disponível quando a conversa ainda está ativa.

    Para isto, deve-se clicar em "Informações do Usuário", em seguida clicar em "Fechar". Em seguida, será solicitado que você adicione um comentário, que serve para registrar o status de conclusão do atendimento.

    Imagem da janela de bate papo inteira, com o item FECHAR circulado


    Marcar como favorito

    Este menu serve para uma conversa como favorita. Ela ficará destacada/em evidência na lista de atendimentos realizados.

    Imagem da janela de bate papo inteira, com o item FAVORITO circulado


    Ultimos Chats

    Este menu exibe a lista de todas as últimas conversas realizadas com este contribuinte, por data e hora. Ao clicar nas datas, será exibida a conversa realizada naquela sessão.

    Imagem da janela de bate papo inteira, com o item ÚLTIMOS CHATS circulados


    Vídeo Chat

    Este menu tem a finalidade de criar um link para bate papo via Áudio e Vídeo, através do sistema JITSI, para chamada como vídeo conferência. Pode servir para confirmar a identidade do contribuinte ou resolver algo mais específico e importante.

    Imagem da janela de bate papo inteira, com o item VÍDEO CHAT circulado


    Preferência de notificações

    Este menu tem como finalidade configurar alertas sobre novas chamadas de contribuintes a serem atendidos. Você pode ativar ou desativar estes alertas.

    Imagem da janela de bate papo inteira, com o item PREFERÊNCIA DE NOTIFICAÇÕES circulados
